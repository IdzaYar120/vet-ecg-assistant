## 📖 Інструкція користувача

Бот працює у режимі діалогу. Ось типовий сценарій використання:

### 1. Початок роботи
Натисніть `/start` або надішліть команду. Бот привітається та запропонує меню.

### 2. Реєстрація пацієнта
Для коректного визначення норм (нормосистолія/тахікардія/брадикардія) боту потрібно знати контекст:
1.  **Оберіть вид тварини:** 🐱 Кіт або 🐶 Собака.
2.  **Введіть вагу:** Напишіть вагу в кг (наприклад: `5.5` або `30`).
    * *Чому це важливо?* Норма пульсу для чихуахуа (140 уд/хв) є тахікардією для лабрадора. Бот це враховує.

### 3. Вибір методу аналізу
Бот запропонує два варіанти:

#### 📸 Спосіб А: Аналіз по фото (Автоматичний)
Ідеально для швидкого скринінгу ритму та пошуку патологій.
1.  Натисніть **"По фото"**.
2.  Надішліть фотографію стрічки ЕКГ (або скріншот монітора).
    * *Порада:* Фотографуйте горизонтально, уникайте сильних відблисків.
3.  Бот запитає: **"Скільки секунд на цьому записі?"**.
    * Порахуйте великі клітинки (при 25 мм/с: 5 великих клітинок = 1 сек). Введіть число (наприклад `3`).
4.  Отримайте розширений звіт з графіком.

#### 📏 Спосіб Б: Аналіз по клітинках (Ручний/Точний)
Використовується, коли фото розмите або потрібна математична точність.
1.  Натисніть **"По клітинках"**.
2.  Оберіть швидкість запису стрічки (25 мм/с або 50 мм/с).
3.  Порахуйте кількість **маленьких клітинок (мм)** між двома сусідніми зубцями R.
4.  Напишіть це число боту.

### 📊 Як читати звіт

Бот генерує картку пацієнта, яка містить:
* **ЧСС:** Поточний пульс та статус (Норма/Бради/Тахі) відповідно до ваги.
* **Ритм:** Регулярний чи хаотичний (з урахуванням синусової аритмії у собак).
* **ШІ Аналіз:** Класифікація типу ударів (N - норма, V - шлуночкова екстрасистола, S - надшлуночкова).
* **Попередження:** Спеціальні алгоритми шукають специфічні патерни:
    * 🧪 *Гіперкаліємія* (високі зубці T).
    * ⚠️ *Електрична альтернація* (підозра на тампонаду).
    * ❤️ *Бігемінія*.

### ⚡️ Корекція результату
Якщо ви помилилися з часом або швидкістю запису, під звітом будуть кнопки швидкої дії:
* `⚡️ Це 50 мм/с` — миттєво перерахувати пульс (х2), якщо забули, що стрічка швидка.
* `🔄 Змінити тварину` — перемкнути норми Кіт/Собака без повторного завантаження фото.
